# 휴먼교육센터 강의 내용 저장용 레포지토리
- 강의 내용 일부 저장

## 단축키(VSCode)

**<span style="font-size: 18px; color: deepskyblue;">[VSCode 단축키 튜토리얼](https://demun.github.io/vscode-tutorial/shortcuts/)</span>**
<hr>
<span style="font-size: 17px;">: 일반 단축키 :</span>

- Ctrl + Shift + P or F1 : 명령팔레트 열기
- Ctrl + P : 파일 열기
- Ctrl + , : 사용자 설정(Settings) 열기
- Ctrl + Shift + N : 새 창 열기
- Ctrl + Shift + W : 현재 창 닫기
- Ctrl + K Ctrl + S : 바로가기 단축키(Keyboard Shortcuts) 목록 보기
